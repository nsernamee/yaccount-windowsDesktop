# YAccount UI 重设计提示词

本文档为 AI 重设计 YAccount 应用 UI 提供的完整需求说明。

---

## 项目概述

- **项目名称**：YAccount - 智能记账应用
- **类型**：Flutter 移动应用（Android/iOS）
- **核心功能**：本地记账、预算管理、数据统计、导入导出
- **目标用户**：个人用户，需要简单、安全的记账工具

---

## 页面结构

应用共 7 个页面，采用底部导航栏切换：

1. **首页 (HomePage)** - 仪表盘，快速记账
2. **历史记录 (HistoryPage)** - 交易列表
3. **统计 (StatisticsPage)** - 数据可视化
4. **预算 (BudgetPage)** - 预算管理
5. **设置 (SettingsPage)** - 应用设置
6. **导入导出 (ImportExportPage)** - 数据管理
7. **添加交易 (AddTransactionPage)** - 底部弹窗表单

---

## 页面详细设计要求

### 1. 首页 (HomePage)

#### 位置
底部导航栏第 1 个 tab

#### 功能模块

**1.1 顶部区域**
- 应用 logo/名称（左侧）
- 货币选择器（右上）：点击弹出货币列表，支持人民币¥、美元$、欧元€等
- 导入导出快捷入口（右上）：点击跳转导入导出页面

**1.2 本月结余卡片**
- 标题："本月结余"
- 金额：大字体显示（收入-支出）
- 负数时金额显示红色
- 超过万亿时使用 Y 单位显示

**1.3 快速记账按钮**
- 两个并排大按钮：支出（红色）、收入（绿色）
- 点击弹出底部记账弹窗

**1.4 预算进度条（如果有设置预算）**
- 显示本月已花费 / 总预算
- 进度条 + 百分比
- 超预算时进度条变红

**1.5 收支概览**
- 默认收起状态，点击展开
- 显示：今日支出/收入、本周支出/收入
- 展开后额外显示：本月支出/收入、本年支出/收入

---

### 2. 添加交易页面 (AddTransactionPage)

#### 类型
底部弹窗（Modal Bottom Sheet）

#### 位置
从首页"记一笔"按钮弹出

#### 功能模块

**2.1 顶部标题栏**
- 标题："添加记录"
- 关闭按钮（右侧）

**2.2 类型切换**
- 两个并排按钮：支出 / 收入
- 选中态有背景色区分（红色=支出，绿色=收入）
- 切换时自动重置分类默认项

**2.3 金额输入**
- 数字键盘，支持小数点
- 货币符号前缀（根据首页选择的货币）
- 金额字体加大加粗

**2.4 日期选择**
- 默认今天
- 点击弹出日期选择器
- 显示格式：2024年1月15日

**2.5 分类选择**
- 网格布局，每行 4 个
- 支出分类：餐饮🍔、出行🚗、购物🛍️、居家🏠、娱乐🎮、医疗💊、教育📚、其他📌
- 收入分类：薪资💰、理财💎、兼职💼、红包🧧、其他📌
- 每个分类有图标 + 名称
- 选中态有背景色

**2.6 备注输入**
- 多行文本输入
- 可选填
- 占位符："添加备注..."

**2.7 保存按钮**
- 底部固定
- 宽度满屏
- 颜色与类型一致（支出红，收入绿）
- 点击保存并关闭弹窗

---

### 3. 历史记录页面 (HistoryPage)

#### 位置
底部导航栏第 2 个 tab

#### 功能模块

**3.1 顶部筛选栏**
- 类型筛选按钮：全部 / 支出 / 收入
- 日期筛选按钮：点击弹出年月选择器
- 筛选激活时按钮上有小红点提示

**3.2 交易列表**
- 按日期分组显示，每组顶部显示日期（如"今天"、"昨天"、"1月14日"）
- 每条记录显示：
  - 分类图标（圆形背景）
  - 分类名称
  - 备注（如果有，超出部分省略号）
  - 金额（支出-红色，收入+绿色）
  - 眼睛图标：点击展开详情
- 记录左滑：显示编辑✏️、删除🗑️ 两个快捷按钮

**3.3 展开详情**
- 点击眼睛图标展开
- 显示：金额、类型、分类、日期、备注

**3.4 空状态**
- 图标 + 文字："暂无记录，添加你的第一笔账目吧"

**3.5 加载更多**
- 滚动到底部自动加载
- 底部显示加载指示器

---

### 4. 统计页面 (StatisticsPage)

#### 位置
底部导航栏第 3 个 tab

#### 功能模块

**4.1 视图切换**
- 两个标签：月份视图 / 年份视图
- 选中态有背景色和阴影

**4.2 日期选择器**
- 渐变色背景卡片
- 左右箭头切换上/下月（或年份）
- 中间显示当前日期（如"2024年1月"）

**4.3 饼图 - 支出分类**
- 卡片容器
- 标题："支出分类"
- 左侧：饼图，显示占比百分比
- 右侧：分类列表（颜色点 + 分类名）
- 无数据时显示"暂无数据"

**4.4 柱状图 - 收支对比**
- 标题："近6个月收支对比"（月份视图）或 "年度12个月收支对比"（年份视图）
- 柱状图：蓝色柱=收入，红色柱=支出
- 底部标签：月份
- 底部图例说明

**4.5 折线图 - 趋势**
- 标题："本月每日支出趋势"（月份视图）或 "年度月度支出趋势"（年份视图）
- 折线图：单色，带底部填充区域
- X轴：日期或月份
- Y轴：金额

---

### 5. 预算页面 (BudgetPage)

#### 位置
底部导航栏第 4 个 tab

#### 功能模块

**5.1 顶部月份选择器**
- 白色卡片
- 左右箭头切换月份
- 中间显示："2024年1月"

**5.2 总预算卡片**
- 渐变色背景（紫蓝）
- 标题："总预算"
- 金额：大字体，未设置显示"未设置"
- 已启用时显示：
  - 右侧百分比标签
  - 底部进度条
  - 底部文字："已花费 ¥xxx，剩余 ¥xxx"
- 编辑按钮（点击修改金额）

**5.3 分类预算列表**
- 标题："分类预算"
- 每个分类预算卡片：
  - 分类名称
  - 百分比标签（颜色根据超预算程度：绿/黄/红）
  - 金额 + 编辑按钮 + 删除按钮
  - 进度条
  - 底部显示："已花费 ¥xxx，剩余 ¥xxx"
- 空状态："点击右下角按钮添加分类预算"

**5.4 添加按钮**
- 右下角 FAB（浮动按钮）
- 点击弹出添加预算对话框

**5.5 添加/编辑预算对话框**
- 金额输入框
- 分类选择下拉框（如果是分类预算）
- 取消 / 确认按钮

---

### 6. 设置页面 (SettingsPage)

#### 位置
底部导航栏第 5 个 tab

#### 功能模块

**6.1 安全设置区块**
- 标题："安全设置"
- 数据库加密开关
- 加密说明卡片：
  - 使用 AES-256 加密
  - 密码用于生成密钥，请妥善保管
  - 忘记密码无法恢复
  - 加密后仍本地存储
- 启用加密时弹出设置密码对话框
- 关闭加密时弹出确认对话框

**6.2 数据管理区块**
- 标题："数据管理"
- 清空所有数据按钮（红色警告色）
- 点击弹出二次确认对话框

**6.3 关于区块**
- 标题："关于"
- 版本号：1.2.0
- 关于 YAccount 按钮
- 点击弹出关于对话框：
  - 应用图标 + 名称
  - 版本号
  - 特点介绍

---

### 7. 导入导出页面 (ImportExportPage)

#### 位置
从首页右上角按钮进入（独立页面，非底部导航）

#### 功能模块

**7.1 顶部导航栏**
- 标题："导入导出"
- 返回按钮

**7.2 导出区块**
- 标题："导出数据"
- 导出为 CSV：图标 + 说明
- 导出为 Excel：图标 + 说明
- 点击后弹出保存/分享选择
- 导出文件名格式：`YYYYMMDD_账本导出.csv` 或 `YYYYMMDD_账本导出.xlsx`
- 文件保存到手机下载目录

**7.3 导入区块**
- 标题："导入数据"
- 从 CSV 导入
- 从 Excel 导入
- 点击选择文件后弹出：
  - 增量追加
  - 覆盖替换（红色警告）

**7.4 危险操作区块**
- 标题："危险操作"（红色）
- 清空所有数据按钮

## 设计规范

### 颜色方案

| 用途 | 颜色 |
|------|------|
| 主色（绿色系） | #00B894 |
| 支出 | #E17055 |
| 收入 | #00B894 |
| 背景 | #F8F9FA |
| 文字主色 | #2D3436 |
| 文字次色 | #636E72 |
| 预算警告 | #FDCB6E |
| 预算超支 | #E17055 |

### 间距与圆角

- 页面内边距：16px
- 卡片圆角：12-16px
- 按钮圆角：8-12px
- 元素间距：8-16px

### 字体

- 标题：18-24px，加粗
- 正文：14-16px
- 辅助文字：12-14px

## 技术约束

- 使用 Flutter 3.x + Dart 3.x
- 使用 Provider 进行状态管理
- 页面数据通过 Provider 获取
- 组件应保持响应式适配
